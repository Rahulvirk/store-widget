1. Overview

The CDI Auction Widget is a responsive, feature-rich auction display component that can be easily integrated into any website. It provides a carousel-style interface for displaying auction items with countdown timers, bid buttons, and quick view functionality.

3. Basic Implementation

Container Requirements

The widget requires a container element with a specific ID. By default, it looks for an element with ID cd-ondemand-container.

Important: The container ID must be unique on your page and must exist before the script loads.

Script Loading

The widget script should be loaded after the container element exists in the DOM. The script will automatically:

  • Detect the container element
  • Initialize the widget with default settings
  • Fetch auction data (mock data by default)
  • Render the carousel interface

Automatic Initialization

The widget automatically initializes when the DOM is ready. No additional JavaScript code is required for basic functionality.

5. API Integration

API Data Format

The widget expects auction data in a specific JSON format. Here's the required structure:

{
  "jsStatus": "success",
  "items": [
    {
      "id": 5551964,
      "title": "Vintage Collectible Sports Memorabilia",
      "url": "https://example.com/auction/5551964",
      "currentBid": "87.07",
      "price": "100.00",
      "closeTime": "Sep 26, 2025, 7:29:00 PM",
      "imgMedium": "https://example.com/images/item1.jpg",
      "imgMedium2": "https://example.com/images/item1-2.jpg",
      "actionTitle": "Bid Now",
      "currentBidTitle": "Current Bid",
      "desc": "Detailed description of the item...",
      "categoryName": "Sports",
      "bids": "10"
    }
  ]
}

Required Fields

  • id - Unique identifier for the auction item
  • title - Item title/name
  • url - Link to the auction page

Optional Fields

  • currentBid - Current highest bid amount
  • price - Starting price (used if currentBid is not available)
  • closeTime - Auction end time (enables countdown timer)
  • imgMedium - Primary item image URL
  • imgMedium2, imgMedium3 - Additional images for carousel
  • actionTitle - Button text (defaults to "Bid Now")
  • currentBidTitle - Label for bid amount (defaults to "Current Bid")
  • desc - Item description (shown in quick view)
  • categoryName - Item category
  • bids - Number of bids placed
Note: If closeTime is not provided or invalid, the countdown timer will not be displayed for that item.

6. Features & Functionality

Responsive Design

The widget automatically adapts to different screen sizes:

  • Desktop (1320px+): 4 items per view
  • Large Tablet (950px-1320px): 3 items per view
  • Tablet (640px-950px): 2 items per view
  • Mobile (640px and below): 1 item per view

Auto-scroll Functionality

The carousel automatically rotates through items with the following behavior:

  • Pauses when user hovers over the widget
  • Pauses when user interacts with navigation
  • Resumes after 3 seconds of inactivity
  • Can be toggled on/off via the indicator button

Touch & Gesture Support

Mobile-friendly features include:

  • Swipe left/right to navigate between items
  • Touch-friendly navigation buttons
  • Responsive touch targets

Quick View Modal

Enhanced item viewing with:

  • Larger item images
  • Image carousel for multiple photos
  • Detailed item information
  • Keyboard navigation (arrow keys)
  • Click outside to close

Countdown Timers

Real-time countdown functionality:

  • Updates every second
  • Shows days, hours, minutes, and seconds
  • Displays "Auction Ended" when time expires
  • Only shows for items with valid closeTime

7. Styling & Customization

CSS Isolation

The widget uses scoped CSS with the .cdi-auction-widget-container prefix to prevent conflicts with your existing styles.

Color Customization

All colors can be customized through the configuration options:

  • dotColor - Navigation dots and arrows
  • counterBgColor - Countdown timer background
  • counterFontColor - Countdown timer text
  • buttonBgColor - Bid button background
  • buttonFontColor - Bid button text

Layout Customization

  • borderRadius - Controls corner rounding for all elements
  • widgetTitle - Customizable main title
  • widgetSubtitle - Customizable subtitle
  • bidButtonText - Customizable button text

Component Visibility

You can control which components are displayed:

  • showBidButton - Show/hide bid buttons
  • showDots - Show/hide carousel dots
  • showArrows - Show/hide navigation arrows
  • enableQuickView - Enable/disable quick view modal

8. Troubleshooting

Common Issues

Widget Not Displaying

Problem: Widget doesn't appear on the page

Solution: Check that the container element exists and has the correct ID before the script loads.

No Auction Data

Problem: Widget shows "No auction listings available"

Solution: Verify your API endpoint is working and returns data in the correct format.

Countdown Timers Not Working

Problem: Countdown timers show "--" or don't update

Solution: Ensure the closeTime field contains a valid date string.

Images Not Loading

Problem: Item images don't display

Solution: Check that image URLs are accessible and return valid images.

Browser Compatibility

The widget is compatible with:

  • Chrome 60+
  • Firefox 55+
  • Safari 12+
  • Edge 79+
  • Mobile browsers (iOS Safari, Chrome Mobile)

Performance Considerations

  • The widget automatically optimizes for performance
  • Images are loaded with error handling
  • Countdown timers are cleaned up when widget is destroyed
  • Event listeners are properly managed

10. WordPress Implementation

Method 1: Using a Custom Plugin

Create a simple WordPress plugin to easily manage the widget across your site:

<?php
/*
Plugin Name: CDI Auction Widget
Description: Adds the CDI Auction Widget to your WordPress site
Version: 1.0.0
Author: Your Name
*/

// Prevent direct access
if (!defined('ABSPATH')) {
    exit;
}

class CDIAuctionWidget {
    
    public function __construct() {
        add_action('wp_enqueue_scripts', array($this, 'enqueue_scripts'));
        add_shortcode('cdi_auction_widget', array($this, 'widget_shortcode'));
        add_action('wp_footer', array($this, 'add_widget_script'));
    }
    
    public function enqueue_scripts() {
        // Enqueue the widget script
        wp_enqueue_script(
            'cdi-auction-widget',
            plugin_dir_url(__FILE__) . 'cd-on-demand.min.js',
            array(),
            '1.0.0',
            true
        );
    }
    
    public function widget_shortcode($atts) {
        $atts = shortcode_atts(array(
            'container_id' => 'cd-ondemand-container',
            'title' => 'Live Auctions',
            'subtitle' => 'Swipe through exclusive items that are ending soon'
        ), $atts);
        
        return '<div id="' . esc_attr($atts['container_id']) . '"></div>';
    }
    
    public function add_widget_script() {
        // Add any additional initialization if needed
        echo '<script>
            // Widget will auto-initialize
            console.log("CDI Auction Widget loaded");
        </script>';
    }
}

// Initialize the plugin
new CDIAuctionWidget();
?>
